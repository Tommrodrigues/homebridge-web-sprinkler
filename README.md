# homebridge-web-sprinkler _(Under Development)_
Homebridge plugin for a web-based sprinkler system

```
"accessories": [
    {
       "accessory": "WebSprinkler",
       "name": "Sprinklers",
       "apiroute": "http://google.com"
     }
]
```

`npm install -g homebridge-web-sprinkler`
